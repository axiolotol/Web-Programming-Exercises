HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

HTML Tags

HTML tags are the building blocks of HTML pages. Here are some essential HTML tags you should know:

<h1> to <h6>

Heading tags define headings of different levels, with h1 being the most important.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>
<p>

Defines a paragraph of text.

<p>This is a paragraph of text.</p>
<a>

Creates a hyperlink to another webpage or resource.

<a href="https://example.com">Visit Example</a>
<img>

Embeds an image in the HTML document.

<img src="image.jpg" alt="Description of image">
<ul> and <ol>

Create unordered (bullet) and ordered (numbered) lists.

<ul>
    <li>Unordered list item 1</li>
    <li>Unordered list item 2</li>
</ul>

<ol>
    <li>Ordered list item 1</li>
    <li>Ordered list item 2</li>
</ol>
<div>

A generic container for grouping content. Used extensively for layout and styling.

<div class="container">
    <!-- Content goes here -->
</div>
<span>

An inline container for text and other inline elements.

<p>This is <span class="highlight">important</span> text.</p>
<form>

Creates an HTML form for user input.

<form action="/submit" method="post">
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>
<table>

Creates a table to display data in rows and columns.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>
<pre> and <code>

Used for displaying preformatted text and code snippets.

<pre>
    <code>
    function greet() {
        console.log("Hello, world!");
    }
    </code>
</pre>

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. CSS can be applied inline, in the head section, or preferably in an external stylesheet.

CSS Syntax

selector {
    property: value;
    another-property: value;
}

Types of Selectors

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#header {
    font-size: 24px;
}

/* Descendant selector */
nav a {
    text-decoration: none;
}

CSS Properties

CSS properties define how HTML elements should be displayed. Here are some essential CSS properties:

color

Sets the color of text.

p {
    color: #333333;
    /* Can use color names, hex, rgb, rgba */
}
background-color

Sets the background color of an element.

.card {
    background-color: #f5f5f5;
}
font-family

Specifies the font for text.

body {
    font-family: 'Arial', sans-serif;
}
font-size

Sets the size of the font.

h1 {
    font-size: 2rem; /* Can use px, em, rem, % */
}
margin & padding

Controls spacing around and within elements.

.box {
    margin: 10px; /* Space outside the element */
    padding: 20px; /* Space inside the element */
    
    /* Can set individual sides */
    margin-top: 15px;
    padding-left: 25px;
}
display

Defines how an element is displayed.

.container {
    display: flex; /* Options: block, inline, flex, grid, none */
}
border

Adds a border around an element.

.frame {
    border: 1px solid #000;
    
    /* Can set individual properties */
    border-width: 2px;
    border-style: dashed;
    border-color: #333;
}
text-align

Controls the horizontal alignment of text.

.center-text {
    text-align: center; /* Options: left, right, center, justify */
}
width & height

Sets the dimensions of an element.

.box {
    width: 300px;
    height: 200px;
    
    /* Can use percentages or other units */
    width: 50%;
    max-width: 600px;
}
position

Specifies the positioning method for an element.

.overlay {
    position: absolute; /* Options: static, relative, absolute, fixed, sticky */
    top: 0;
    left: 0;
}
